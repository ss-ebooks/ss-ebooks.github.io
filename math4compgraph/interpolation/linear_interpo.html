Linear Interpolation

สมมุติว่าต้องการประมาณตำแหน่งของจุด c ซึ่งอยู่ระหว่างจุด a และ b ซึ่งทราบค่าแล้ว โดยกำหนดว่าให้จุด c ห่างจากจุด a เป็นระยะทาง 40% ของระยะทางจากจุด a ไปยัง b การคำนวณหาตำแหน่งของ c อาจทำได้ดังนี้

  1. หาค่าของระยะทางจาก a ไป b Da,b​​=b−a​

  2. หาค่าของระยะทางจาก a ไป c Da,c​Da,c​​=10040​(b−a)=10040​Da,b​​

ในตัวอย่างได้ยกค่า 10040​ ขึนมาเพื่อให้เห็นภาพง่าย แต่ค่านี้สามารถเปลี่ยนแปลงได้ตั้ง 0→1 และหากใช้ t แทนค่านี้ ก็จะเขียน ความสัมพันธ์ใหม่ได้เป็น

Da,c​=tDa,b​

  1. ตำแหน่งของ c ที่แท้จริงจะคำนวณได้จาก

Dc​Dc​​=Da​+Da,c​=Da​+tDa,b​​ หรือ cc∴c​=a+t(b−a)=a+tb−ta=(1−t)a+tb​(1.0)​

ถ้าให้ n0​ คือจุดเริ่มต้น และ n1​ คือจุดสิ้นสุด และ n คือจุดใดๆ ที่อยู่ระหว่าง n0​,n1​ การประมาณค่าของ n เขียนความสัมพันธ์ ให้อยู่ในรูปแบบทัวไปดังนี้

n​=(1−t)n0​+tn1​(1.1)​

เรียกสมการ (1.1) ว่าเป็น interpolant สามารถเขียนในรูปแบบของ matrix ได้ ดังนี้

nn​=[1−t​t​]⋅[n0​n1​​]=[t​1​]⋅[−11​10​]⋅[n0​n1​​]​(1.2)​

เมื่อ t คือ ratio จะมีค่าอยู่ระหว่าง [0,1] หมายความว่า

  1. เมื่อ t=0 ค่าที่ต้องการประมาณค่าจะอยู่ ณ จุด a (จุดเริ่มต้น)
  2. เมื่อ t=1 ค่าที่ต้องการประมาณค่าจะอยู่ ณ จุด b (จุดสิ้นสุด)
  3. 0≤t≤1 ่คาที่ต้องการประมาณค่าจะอยูระหว่างจุด a,b

พิจารณาสมการ (1.1) ในรูปแบบของ polynomial

  1. เป็น polynomial degree 1 (ตัวแปร x มีกำลังสูงสุดเป็น 1)
  2. การประมาณตำแหน่งของจุดอื่นที่อยู่ระหว่าง 2 จุด ทำได้โดยการเปลี่ยนค่าของ coefficient ในกรณีนี้คือ t,(1−t)
  3. ถ้าข้อมูลอยู่ในรูป coordinate สามารถแยกคำนวณในแต่ละ axis ได้ เช่น xyz​=(1−t)x0​+tx1​=(1−t)y0​+ty1​=(1−t)z0​+tz1​​ หรือเขียนในรูปของ matrix [x​y​z​]=[t​1​]⋅[−11​10​]⋅[x0​x1​​y0​y1​​z0​z1​​]

รูปที่ 2 แสดงตัวอย่างการประมาณค่าที่อยู่ระหว่าง 1 กับ 5 ตามสมการ (1.1) จะเห็นได้ชัดเจนว่าการเพิ่มขึ้นของค่าที่ถูกประมาณจะเพิ่มขึ้นเป็นเส้นตรงเมื่อเทียบกับค่าของ t

นำไปใช้กับข้อมูล 2 มิติ โดยให้จุด n0​=(1,1) และ n1​=(4,5) ผลของการทำ linear interpolation ออกมาเป็นเส้นตรง ดังรูปที่ 3 แสดงว่าสมการ (1.2) สามารถใช้ได้กับข้อมูลทีมีหลายมิติได้
