Non-linear Interpolate

1. Trigonometry Interpolate

จาก Pythagorean identity ทราบว่า sin2(θ)+cos2(θ)​=1​ เมื่อ 0≤θ≤2π​

จะเห็นได้ว่าเมื่อนำไปเทียบกับ t+(1−t)=1 จากเรื่อง linear interpolation จะเหมือนกัน ดังนั้นเราจึงสามารถนำ trigonomety identity นี้มาใช้ ในการทำ interpolation ได้เช่นกัน โดยเทียบจากสมการ (1.1)

n=n0​sin2(θ)+n1​cos2(θ)​(2.0)​ เมื่อ 0≤θ≤2π​

รูปที่ 1 แสดงกราฟของค่า n ที่ได้จากการทำ trigonometry interpolation เมื่อกำหนดให้ n0​=1,n1​=5 เมื่อ 0≤θ≤2π​ จะเห็นได้ว่าไม่ได้เป็นเส้นตรง (เส้นสีน้ำเงิน)

ถ้าทดลองทำใน 2 มิติเช่น จุดเริ่มต้นที่ (1,1) และจุดสิ้นสุดที่ (5,4) ผลที่ได้แสดงในรูปที่ 2 ซึ่งแสดงให้เห็นรูปแบบของเส้นตรง หมายความว่า trigometry interpolation จะเหมาะกับข้อมูลที่มีมีติเดียว แต่ถ้ามีมิติมากขึ้นจะแสดงผลออกมาเช่นเดียวกับ linear interpolation

2.3. Spline Interpolation

"Spline" มาจากคำใช้เรียกไม้บรรทัดแบบที่งอได้เพื่อใช้ในการวาดเส้นโค้ง ดังภาพที่แสดงข้างล่าง

Spline interpolation อาจถูกจัดอยู่ในกลุ่ม polynomial interpolation ได้ เพราะให้ผลลัพธ์ที่เหมือนกัน ต่างกันตรงที่ แทนที่จะสร้าง polynomial ชุดเดียว แต่ spline interpolation ใช้การสร้าง polynomial degree มาประมาณค่าข้อมูลทีละช่วงข้อมูล (piecewise function) ทำให้เกิด polynomial มากกว่า 1 แม้จะมี polynomial มากแต่ประโยชน์ที่ตามมาคือ polynomial จะมี degree ต่ำ (2-3) ซึ่งจะเป็นการลด Runge phenomenon ที่มักจะเกิดขึ้นมาใช้ polynomial degree สูงขึ้นในการทำ interpolation

ถ้ามีข้อมูลอยู่ n+1 จำนวน (x0​,y0​),(x1​,y1​),(x2​,y2​),...,(xn​,yn​) โดย x0​≤x1​≤x2​≤x3​≤...≤xn​ แล้ว จำนวน piecewise functions ที่จะมีได้คือ n ดังนี้

S(x)=⎩⎨⎧​s0​(x),s1​(x),s2​(x),...sn−1​(x),​x0​≤x≤x1​x1​≤x≤x2​x2​≤x≤x3​xn−1​≤x≤xn​​

ยกตัวอย่าง ถ้ามีชุดข้อมูล 3 จำนวนคือ (2,1),(5,8),(7,3) จำนวน piecewise function ที่จะเกิดขึ้นคือ

  • s0​(x) ใช้ประมาณค่าข้อมูลระหว่าง 2≤x≤5
  • s1​(x) ใช้ประมาณค่าข้อมูลระหว่าง 5≤x≤7

2.3.1 Boundary condition

เป็นเงือนไขที่กำหนดขึ้นเพื่อให้สามารถทำการคำนวณหา coefficients ของ piecewise functions ได้ เงื่อนไขมีดังนี้

  1. Natural spline กำหนดว่า 2nd derivatives ของ function แรก s0​(x) และ function สุดท้าย sn−1​(x) จะมีค่าเป็น 0 จุดเชื่อมของ functions s0′′​(x0​)sn−1′′​(xn−1​)​=0=0​

  2. Not a knot spline กำหนดว่า 3rdderivative ของ 2 function แรก มีค่าเท่ากัน s0′′′​(x0​)=s1′′′​(x1​)

  3. Periodic spline s0′​(x0​)s0′′​(x0​)​=sn−1′​(xn−1​)=sn−1′′​(xn−1​)​

  4. Quadratic spline a0​=an−1​=0
