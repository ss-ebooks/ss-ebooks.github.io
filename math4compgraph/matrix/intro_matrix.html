Matrix

ถ้าเรานำตัวเลขมาจัดรูปแบบคล้ายตาราง เราเรียกสิ่งนั้นว่า matrix เช่น ​321​201​​ โดยมีข้อที่ควรทำความรู้จักเบื้องต้นดังนี้

  1. matrix อาจถูกแทนที่ด้วยอักษรภาษาอังกฤษ ซึ่งมักจะใช้ตัวพิมพ์ใหญ่ เช่น A=​321​201​​

  2. คุณสมบัติหนึ่งที่สำคัญของ matrix คือขนาด (size, dimension) โดยจะบอกเป็นค่าของจำนวนแถวและคอลัมน์ จากตัวอย่างข้างต้นเป็น matrix ที่มีขนาด 3 แถว 2 คอลัมน์ เขียนเป็น A3×2

  3. ตัวเลขแต่ละตัวที่อยู่ใน matrix เรียกว่า element หรือ entry ่จากตัวอย่าง matrix A ประกอบด้วย 6 elements (entries)

  4. Element แต่ละตัวจะมีตำแหน่งเฉพาะตัว โดยระบุเป็นค่าของแถว (row) และ คอลัมน์ (column) อาจเริ่มนับจากศูนย์หรือหนึ่งก็ได้ (การนับที่เริ่มจาก 0 จะพบในการเขียนโปรแกรมในบางภาษา เช่น C,C++) ถ้าให้ arc​ แทน element ของ A อาจเขียนได้ดังนี้ a1,1​a1,2​a2,1​a2,2​a3,1​a3,2​​=3=2=2=0=1=1​
