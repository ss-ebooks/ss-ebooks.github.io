Motivation

Python คือ hight level computer programming language ที่มีรูปแบบการสื่อสารใกล้เคียงกับภาษาเขียนในภาษาอังกฤษ
มี data structures พร้อมใช้งาน (built in) ใช้การแปลภาษาแบบ interpreter ทำให้ใช้งานง่ายแก่ผู้เริ่มต้นและเหมาะกับการสร้างโปรแกรมต้นแบบ
นอกจากนี้ยังสามารถเชื่อมต่อไปใช้งานร่วมกับภาษาอื่น เช่น C, Go ได้อีกด้วย งานที่นิยมนำ Python ไปใช้งานได้แก่

"Statement" คือชุดคำสั่งที่เขียนในโปรแกรมเพื่อจะนำไปสั่งการคอมพิวเตอร์
"Syntax" คือกฏในการเขียนชุดคำสั่ง แต่ละ computer language จะมี syntax ต่างกัน ทำนองเดียวกับภาษามนุษย์ที่มีกฏการใช้ภาษา ต่างกันตรงที่ การผิดกฏในภาษามนุษย์ยังคงใช้สื่อสารได้ ในขณะที่ computer language จะไม่อนุญาตให้มีการทำผิด syntax เลย

Code block

Python program ประกอบขึ้นจาก code block ตั้งแต่ 1 code block ขึ้นไป แต่ละ code block ประกอบด้วย statement ตั้งแต่ 1 statement ขึ้น ที่สร้างขึ้นมาเพื่อทำงานใน 1 หน่วยของงาน

การย่อหน้าและการแบ่งบรรทัด (indentation and line feed)

computer programming language หลายภาษาใช้ "{}" เพื่อให้ทราบถึงการแยกกลุ่ม statement หรือ code block สำหรับ Python ใช้การย่อหน้า (indentation) ซึ่งเป็นลักษณะเฉพาะของ Python


จากภาพ กล่องแต่ละสีเปรียบได้กับ 1 code block กล่องเหล่านี้ถูกวางเยื้องกัน เทียบได้กับ code block ที่มีย่อหน้าต่างกันจะถือว่าอยู่กันคนละ code block ภายในแต่ละ code block จะต้องมี statement อย่างน้อย 1 statement การแยกแต่ละ statement ออกจากกันในภาษา Python ใช้การขึ้นบรรทัดใหม่ (line feed) (เมื่อกดแป้น Enter)


ตัวอย่าง Python code ข้างบนประกอบด้วย 4 code blocks ลำดับการทำงานตามลำดับดังนี้

  1. block 1 จะถูกประมวลผลก่อน
  2. block 2 ถูกประมวลผลตามมา
  3. เมื่อเข้ามาใน block 2 แล้ว block 3 จะถูกนำไปประมวลผล
  4. ถายใน block 3 มี condition if หากเงื่อนไขเป็นจริง block 4 จะถูกเรียกใช้งาน หากเป็นเท็จ block 4 จะไม่ถูกเรียก
การใช้โครงสร้างแบบนี้ อาจทำให้เกิดความยุ่งยากสำหรับนักพัฒนาที่ต้องการกำหนดโครงสร้างแบบที่ตัวเองต้องการ แต่ก็ทำให้เกิดข้อดี 3 ข้อ
  1. การอ่าน code จะเห็นการจบบรรทัดที่ชัดเจนว่าอยู่ตรงไหน
  2. โครงสร้างของ code จะเหมือนกันไม่ว่าจะเปลี่ยนคนเขียนชุดคำสั่ง
  3. ช่วยให้การอ่านชุดคำสั่งง่ายขึ้น


ในบางครั้งอาจต้องมีการเขียุนชุดคำสั่งที่ยาวมากจนการเขียนไว้ในบรรทัดเดียวอาจไม่สะดวกในการอ่าน สามารถแยกชุดคำสั่งออกไปยังบรรทัดใหม่โดยที่ยังคงความต่อเนื่องได้โดยการใช้ "\" 

			 
a = 20
b = 30
c = 15
if (a != b) and (b != c) \
    and (c!=a) :
    print("This is an example.")

 Tips:

  • Python program ประกอบด้วย code block อย่างน้อย 1 code block
  • Code block คือกลุ่มของ statement ที่ทำงาน 1 unit
  • Python ใช้ line feed ในการแบ่ง statement ออกจากกัน และใช้ indentation แยก code block

การเขียนคำอธิบาย (comment)

การให้คำอธิบายมีความสำคัญในการเขียนชุดคำสั่งสำหรับคอมพิวเตอร์โปรแกรม นอกจากเป็นบันทึกช่วยจำได้แล้วยังช่วยให้คนอื่นเข้าใจเหตุผลว่าอะไรคือเหตุผลของการเขียนชุดคำสั่งนั้นขึ้นมา ในการเขียนคำอธิบายที่มีบรรทัดเดียว ใช้ # และหากเขียนหลายบรรทัดใช้ """ หรือ ''' ล้อมรอบ เช่น

When you want to write long comment
you should surrond  """ or '''
