Functions

Motivation

ตัวอย่างประโยคทางคณิตศาสตร์ เช่น \(f(x,y) = x^2 + y^2\) บอกเราว่า f คือ function ที่โยงค่าจำนวนจริง 2 จำนวนแทนด้วย x,y ไปยังจำนวนจริงอีกจำนวนที่มีค่าเป็น \( x^2 + y^2\) ถ้าเรานำเข้าตัวเลข \( x=1,y=2 \) สู่ function f จะได้เป็น \( f(1,2) = 5 \)


การทำงานของคอมพิวเตอร์โปรแกรมก็มีหลักการทำงานเหมือนกัน และมักต้องอาศัยการทำงานหรือชุดคำสั่งบางชุดซ้ำๆ เช่น การคำนวณทางคณิตศาสตร์ แบบตัวอย่างที่กล่าวมา ชุดคำสั่งที่ใช้งานในลักษณะนี้ควรถูกแยกออกมาไว้ต่างหาก เพื่อไม่ต้องเขียนชุดคำสั่งซ้ำๆ ชุคคำสั่งที่ทำงานเฉพาะด้านโดยอาจจะสงผลลัพธ์ เมื่อถูกเรียกใช้ เรียกว่า"Function"


ส่วนของโปรแกรมที่เป็น function เรียกว่า "function definition" เมื่อต้องการนำ function ที่ถุูกสร้างไว้มาใช้งานจะเรียกว่า "call" หรือ ""invoke" function


Function อาจมีการรับค่าบางอย่างผ่านตัวแปร (input arguments) หรือ ผลลัพธ์ (output parameters) ได้ เช่น function คำนวณค่าเฉลี่ย input arguments อาจเป็นคะแนนจากตัวอย่าง และ output paramters คือ ค่าเฉลี่ยที่จากการนำ input มาบวกกันแล้วหารด้วยจำนวน input เป็นต้น ชุดคำสังภายใน function เรียกว่า "body of the function"


การสร้าง Function

การสร้าง function ในภาษา Python มีมากกว่า 1 วิธี วิธีที่ใช้กันทั่วไปคือ การใช้ keyword "def" เพื่อระบุ funtion definition รูปแบบทั่วไปคือ 


def function_name (argument1, argument2,...) :
	# function description
    statement 1 # statement comment
    statement 2 # statement comment
    ... 
    return output parameters

จากโครงสร้าข้างบน มีองค์ประกอบหลัก 2 ส่วน

1. Header : ประกอบด้วย
  • keyword "def"
  • ชื่อ function ซึ่งใช้หลักการเดียวกับการตั้งชื่อตัวแปร
  • สัญญลักษณ์ "()" ในกรณีที่มี input argument จะนำมาไว้ในนี้
  • สัญญลักษณ์ ":"
2. Function body : ประกอบด้วย
  • description เป็น comment ที่อธิบายการทำงาน การรับค่า input การส่งค่า output หรืออื่นๆ เพื่อให้เกิดความเข้าใจ ผู้เขียนอาจจะทำหรือไม่ก็ได้
  • ชุดคำสั่ง (function statements)
  • คำอธิบายชุดคำสั่ง ส่วนนี้อาจไม่จำเป็นต้องมี
  • return parameters ส่วนนี้อาจจะมีหรือไม่ก็ได้

ส่วนนี้จะต้องมีการย่อหน้า identation ถัดจากส่วน header


เมื่อสร้าง function เสร็จแล้ว การเรียกใช้งานทำได้โดยการ พิมพ์ชื่อของ function หาก function นั้นต้องการ arguments ก็ต้องให้ค่า arguments ตามลำดับ ดังตัวอย่าง 

# function with return
              
def compute_average(x1,x2,x3,x4):
  """
  To do:            
  This function computes average of 4 numbers given by users.
              
  Input arguments:              
  x1,x2,x3 and x4 could be either floats or integers.
              
  Output :
  return average in term of float.              
  
  """              
  sum = x1 + x2 + x3 + x4 # compute summation
  mean = sum / 4 # compute average
  
  return mean

# call function
v1 = 2
v2 = 3
v3 = 5
v4 = 5
m = compute_mean(v1,v2,v3,v4)
print(f"Mean is {m}")    

# function with no return  
def print_name(name):
  print(f"Your name is {name}")

 Tip:

  • การตั้งชื่อ function ที่ดีควรใช้ชื่อที่สื่อความหมายได้ แต่ไม่ควรยาวเกินไป ใช้ _ แทนการเว้นวรรค และนิยมใช้อักษรตัวเล็ก

Default value of argument

Argument ที่ส่งให้ function สามารถถูกกำหนดค่าไว้ล่วงหน้าหรือค่าปริยายได้ เช่น

def greeting(name,msg="Good morning"):
	print(f"{msg} {name}")

function greeting ต้องการ argument 2 ตัวคือ name และ msg โดย msg ได้มีการกำหนด default value เป็น "Good morning" ในขณะที่ name ไม่ถูกกำหนด argument ที่ไม่มี default value จะต้องถูกกำหนดค่าทุกครั้ง (mandatory) เมื่อ function ถูกเรียกใช้ ในขณะที่ argument ที่มี default value อาจไม่ต้องถูกกำหนด ค่าให้ก็ได้ หากไม่มีการกำหนดค่าให้ตอน function ถูกเรียกใช้ ค่า default จะถูกนำมาใช้งาน เช่น 

who = "Sam"
msg = "Good afternoon"

greeting(who)
greeting(who,msg)

แบบแรกเป็นการเรียกใช้ function โดยใช้แต่ mandatory argument ซึ่ง default value จะถูกเรียกใช้ และแบบที่ 2 เป็นการกำหนดค่าทับ default value ผลลัพธ์ที่ได้คือ

Good morning Sam
Good afternoon Sam

  Remember:

จะต้องกำหนด mandatory argument ทั้งหมดเสียก่อนจึงจะประกาศ default value argument ตัวอย่างแบบที่ถูกต้อง เช่น 

def greeting(name,msg="Good morning"):
	print(f"{msg} {name}")

แบบที่ไม่ถูกต้อง เช่น 

def greeting(msg="Good morning",name):
	print(f"{msg} {name}")

Local and Global Variables

จากที่กล่าวไปก่อนหน้าว่าตัวแปรจะโยงกับตำแหน่งของข้อมูลที่จัดเก็บในหน่วยความจำคอมพิวเตอร์ การใช้ function ก็มีการกำหนดตำแหน่งในหน่วยความจำสำหรับตัวแปรของตัวเองเช่นกัน ตัวแปรที่ใช้ภายใน function ใดๆจะไม่เกี่ยวข้องกัน เรียกตัวแปรแบบนี้ว่า local variable

มีตัวแปรอีกแบบหนึ่งที่ทุก function ในโปรแกรมเดียวกันสามารถจะเข้าถึงได้ เรียกว่า global variable เนื่องจากหน่วยความจำของแต่ละ function จะเปิดก็ต่อเมื่อ function นั้นถูกเรียกใช้งาน หากต้องการส่งค่าไปมาระหว่าง function ก็สามารถนำข้อมูลไปพักไว้ที่ global variable ได้



การประกาศตัวแปรทั้งแบบ global variable และ local variable ใช้วิธีการประกาศตัวแปรตามที่กล่าวมาแล้ว ต่างกันที่ตำแหน่งที่วาง โดย global variable จะต้องอยู่ข้างนอก function ในขณะที่ local variable จะอยู่ใน function 


# This is global variable
var1 = "Hello "
              
def func1():
  print(f"Value of global variable inside function is \"{var1}\" ")
                       
# access vaule outside function
print(f"Value of global variable outside function is \"{var1}\" ")              

# access vaule inside function
func1()

# chage value of global variable
var1 = "I live in Bangkok."

# test 
func1()


จากตัวอย่าง code ข้างบน var1 เป็นตัวแปร global variable เพราะถูกประกาศไว้นอก function และ func1 เป็น function ที่ได้เข้าถึงตัวแปร var1 (บรรทัดที่ 5) ในบรรทัดที่ 8 และ 11 ได้แสดงให้เห็นว่าค่าของ global variable จะมีค่าเท่ากันไม่ว่าจะเข้าถึงจากในหรือนอก function บรรทัดที่ 14 และ 17 แสดงให้เห็นถึงผลจากการเปลี่ยนแปลงค่าของ global variable 


            
# This is global variable
var1 = "Hello "
              
def func1():
  # local variable
  var1 = 10
  print(f"Value of local variable inside function is \"{var1}\" ")

def func2():    
  # access global variable
  print(f"Value of global variable inside function is \"{var1}\" ")              

# access local variable 
func1()
# access local variable               
func2()              
              

# chage value of global variable
var1 = "I live in Bangkok."

# test 
func1()
func2()

อธิบาย

  • บรรทัดที่ 2 มีการประกาศตัวแปรแบบ global ชื่อ var1 มีค่าเป็น "Hello "
  • บรรทัดที่ 6 มีการประกาศตัวแปรแบบ local ชื่อ var1 เหมือนกับ global variable
  • บรรทัดที่ 11 มีการเข้าถึงตัวแปร global ชื่อ var1 ภายใน func2
  • บรรทัดที่ 14,16 เรียก func1 และ func2 ขอสังเกตุว่าผลลัพธ์จะต่างกัน
  • บรรทัดที่ 20 มีการเปลี่ยนค่าของ global variable
  • บรรทัดที่ 23,24 ทดสอบผลลัพธ์


code ตัวอย่างข้างบน แสดงให้เห็นขอบเขตของ global variable ต่างจาก local variable อย่างไร การเปลี่ยนค่าของ local variable จะเกิดขึ้นภายใน function เท่านั้นจะไม่ส่งผลต่อ global variable แม้จะมีชื่อที่เหมือนกันก็ตาม และขอให้สังเกตุว่า function จะเข้าถึง local variable ก่อน global variable ถ้าทั้งสองมีชื่อซ้ำกัน


global keyword

ใช้บอก Python ให้ทราบว่า global variable จะสามารถถูกปรับปรุงค่าได้ภายใน function ที่ใช้ keyword นี้ 

# This is global variable
var1 = 100
              
def func1():
  # try to modify global variable
  var1 = var1 + 5
  print(f"Value of local variable is \"{var1}\" ")

#call function
func1()

ถ้าทำการเรียก func1() จะสังเกตุว่ามีข้อความแสดงความผิดพลาดขึ้น และมีข้อความส่วนหนึ่งเป็น UnboundLocalError: local variable 'var1' referenced before assignment นั่นเพราะมีความพยายามจะปรับค่าของ global variable ใน function (บรรทัดที่ 6) โดยไม่มี keyword global

ทดลองกพิมพ์คำสั่ง global var1 แทรกเข้าไปก่อนบรรทัดที่ 6 แล้วลอง run ชุดคำสั่งอีกครั้ง ผลที่ได้คราวนี้จะไม่พบความผิดพลาดแต่จะได้ผลลัพธ์ที่แสดงว่า var1 ได้เปลี่ยนค่าไปแล้ว


Lambda Function

นีกถึงการเขียน function ทางคณิตศาสตร์ที่ถูกเขียนไว้ในบรรดเดียว เห็นแล้วเข้าใจได้ทันที เช่น

  • สมการหาพื้นที่วงกลมรัศมี r, \(f(r) = \pi r^2\)
  • square sum , \(f(x,y) = x^2+y^2 \)

Python มีการสร้าง function อีกแบบหนึ่งที่มีโครงสร้างคล้ายกับตัวอย่างสมการที่ยกตัวอย่างมา ไม่จำเป็นต้องมีการตั้งชื่อแต่ต้องใช้ keyword lambda เขียนชุคคำสั่งเสร็จสิ้นในบรรทัดเดียว รับ argument ได้เหมือน function ปรกติ มีการ return ค่าทุกครั้งเรียกว่า Lambda function รูปแบบทั่วไปคือ 

lambda argument : expression        
      

# declare lambda function
circle_area = lambda r : 3.414 * r * r
sqr_sum = lambda x,y : ( x * x) + (y * y)      
concat_str = lambda x,y : x+y              

# call lambda function
print(f"Area of a circle with radius 3 units is {circle_area(3)}" )
print(f"Square sum of 8,9 is {sqr_sum(8,9)}" )              
print(concat_str("Ant"," man"))

First class citizen

function ในภาษา Python จัดเป็น first-class citizen หมายความว่าสามารถใช้งานในรูปแบบ นอกจากรูปแบบที่กล่าวมาก่อนหน้าแล้ว ยังสามารถใช้งาน function ในรูปแบบอื่นดังนี้

  1. รับหน้าที่เป็น arguments
  2. ถูก assigned ค่าให้กับตัวแปรได้
  3. สร้างและทำงานภายใน function อื่น (nested function)


Nested function 

          
def parent_function():
  instruction
  
  def inner_function():
    instrunctions
        
  inner_function # call inner_function
  
  return output
            
        

   
def power(base,exponent):
  # declare inner function        
  def compute():
    return base ** exponent

  # call inner function
  result = compute()
  return result
          
print(power(2,3))

Function as argument

Function สามารถทำหน้าที่เป็น function argument ได้ด้วย ดูตัวอย่าง 

def decorator(input_func):
  def wrapper():
    print("Before input function.")
    #call input function
    input_func()
    print("After input function. ")
  wrapper()
  
          
def standalone_func():
    print("This is stand alone function.")
  
# pass function as argument
decorator(standalone_func)

Summary

  1. Function คือชุดคำสั่งที่ถูกสร้างเพื่อใช้งานเฉพาะด้าน อาจรับ argument เข้าหรืออาจ return ผลลัพธ์ได้
  2. Function มีพื้นที่หน่วยความจำของตัวเอง เพื่อใช้สำหรับ local variable และ instruction
  3. Local variable คือตัวแปรที่ใช้งานได้แ่ต่ภายใน function ในขณที่ global variable สามารถเข้าถึงได้จากทุก function ในโปรแกรม
  4. ในกรณีที่ต้องมีการปรับแต่งค่า global variable ภายใน function ต้องใช้ keyword global
  5. lambda function คือ function ที่ไม่ต้องมีชื่อ ใช้ instruction เพียงบรรทัดเดียว
  6. function สามารถถูกใช้เป็น argument ของ function อื่น หรือถูกสร้างภายใน function อื่นได้
