การสร้างโปรแกรม (Writing a program)

ขั้นตอนการพัฒนาซอฟต์แวร์ (Software Development Process)

เนื้อหาในตอนนี้จะกล่าวถึงภาพรวมของขั้นตอนการพัฒนาซอฟต์แวร์ เพื่อใช้เป็นพื้นฐานในการศึกษาในรายละเอียดที่มากกว่าได้ง่ายขึ้น โดยทั่วไปขั้นตอนแบ่งได้เป็น 6 ขั้นตอน


1. วิเคราะห์และทำความเข้าใจปัญหา (Analyze and understand problem)

ทำความเข้าใจว่าอะไรคือปัญหาที่จะแก้ ไม่จำเป็นต้องเข้าใจอย่างถ่องแท้แต่อย่างน้อยก็ให้เห็นภาพรวม


2. กำหนดขอบเขตการทำงาน (Determine specification)

  • กำหนดว่าโปรแกรมที่จะสร้างต้องทำอะไร ไม่จำเป็นต้องกังวลถึงเรื่องขั้นตอนการทำงาน
  • กำหนด input และ output ของโปรแกรม

3. ออกแบบขั้นตอนการแก้ปัญหา (Create a design)

ออกแบบโครงสร้างของโปรแกรม

  • ลงรายละเอียดการทำงาน
  • เลือกใช้หรือสร้าง algorithm

4. การพัฒนาโปรแกรมคอมพิวเตอร์(Implement the design / Create a program)

แปลสิ่งทีออกแบบไว้เป็น computer program ด้วย computer language (ในที่นี้ใช้ Python)


5. ทดสอบ แก้ไข (Test and Debug the program)

  • ทดสอบการทำงานของโปรแกรม
  • "Debugging" คือกระบวนการหาข้อผิดพลาดในโปรแกรมแล้วทำการแก้ไข

6. ดูแล ปรับปรุง(Maintain the program)

ในโลกของความเป็นจริง โปรแกรมที่พัฒนาจนใช้งานได้จริงมักไม่เคยจบสิ้นการดูแลและปรับปรุง เพราะความต้องการมีการเปลี่ยนแปลงได้เสมอ

3. ออกแบบขั้นตอนการทำงาน (Create a design)

เขียนขั้นตอนในการทำงานแบบคร่าว (Psuedocode)

  1. สร้างตารางเทียบระหว่างอักษรและตัวเลขกับ Morse code
  2. แยกคำหรือตัวเลขจากประโยคที่รับจากผู้ใช้ออกเป็นส่วนๆ
  3. นำคำหรือตัวเลขที่ได้จากข้อ 2 ไปเทียบหา Mors code จากตารางในข้อ 1

4.2 การแยกคำ (Split English sentence) function ที่สร้างทำงานโดยนำเข้าประโยค 1 ประโยค เปลี่ยนอักษรให้เป็น capital letter ทั้งหมด แล้วทำการแยกคำออกจากประโยค 

def spliter(sentence):
  # to upper case
  sentence = sentence.upper()
  
  # split 
  word_list = sentence.split(" ")
  
  return word_list

4.3 เทียบคำกับ Morse Code โดย function ที่สร้างจะนำเข้าเป็นคำโดด นำอักษรของคำไปเทียบกับอักษรในตารางทีละอักษรเพื่อให้ทราบถึง Morse code แล้วนำเอา Morse code ที่ได้มาเรียงต่อกันในลำดับเดียวกันกับการเรียงของอักษรในคำ

def Morse_encoder(a_word):	
	result = ""
	for l in a_word:
		mc = morse_lookup[l]
		result = result + mc + " " 

	return result

5. ทดสอบและหาข้อผิดพลาด (Test and debug the program)

5.1 ทดสอบด้วยประโยคสำเร็จ ใช้อักษรภาษาอังกฤษหรืออาจมีตัวเลขอาราบิกปน เพื่อทดสอบการทำงานก่อน

test_sent = "THIS IS MORSE CODE"
words = spliter(test_sent)
result = ""
for w in words :
	result = result + Morse_encoder(w) + " "
print(result)

ผลลัพธ์ที่แสดงบนหน้าจอ บอกว่าการทำงานของโปรแกรมถูกต้อง สามารถใช้งานได้

5.2 ทดสอบการจัดการกับข้อผิดพลาดที่อาจเกิดขึ้นได้ เช่น การเผลอพิมพ์อักษรที่ไม่ใช่ภาษาอังกฤษแทรกในประโยค เช่น 

test_sent = "THIS IS MORSEก CODE"
words = spliter(test_sent)
result = ""
for w in words :
	result = result + Morse_encoder(w) + " "
print(result)

Python interpreter แจ้งข้อผิดพลาดว่า KeyError: ก และโปรแกรหยุดทำงาน ทั้งนี้ เนื่องจากการทำงานของโปรแกรมคือใช้อักษรภาษาอังกฤษหรือตัวเลขเป็น key ในการสืบค้นหา Morse code ที่ตรงกับอักษรนั้นออกมา ถ้าไม่มี key ดังกล่าว interpreter จึงฟ้องข้อผิดพลาดออกมาและหยุดทำงาน การแก้ไขอาจทำได้หลายกรณี แต่ในตัวอย่างจะใช้การข้ามอักษรดังกล่าวไป โดยอาศัยคำสั่ง continue ชุดคำสั่งใหม่ของ Morse_encoder() 

def Morse_encoder(a_word):	
	result = ""
	for l in a_word:
		if l not in morse_lookup.keys() :
			continue
		mc = morse_lookup[l]
		result = result + mc + " " 

	return result

6. บำรุงรักษา ปรับปรุง (Maintain the program)

จากผลลัพธ์ที่เห็นแสดงว่าโปรแกรมสามารถทำงานได้แล้วในภาพรวม แต่ถ้าพิจารณาดูจะพบว่าโปรแกรมยังสามารถพัฒนาต่อได้ในอีกหลายจุด ได้แก่

  • การลดความผิดพลาดจากการข้อมูลของผู้ใช้
  • การแสดงผลที่เข้าใจและนำไปใช้งานง่ายขึ้น
  • การสร้างระบบ User Interface
  • การทำงานเชื่อมโยงไปยังระบบอื่น เช่น ระบบไฟกระพริบผ่านสาย fiber optics
  • ฯลฯ

ในตัวอย่างนี้ขอยกกรณีการปรับปรุงอย่างง่ายคือการสร้างส่วน user interface ใ้ห้ทำงานวนมารับประโยคจากผู้ใช้แล้วพิมพ์ Morse code บนหน้าจอไปเรื่อยๆ จนกว่า ผู้ใช้จะพิมพ์อักษร # เพื่อยุติการทำงาน ชุดคำสั่งใน main() จะถูกปรับปรุงไปเป็น 

def main():	
	test_sent = input("Enter sentence (press Enter to process) >> ")	
	while test_sent != "#" :
		words = spliter(test_sent)
		result = ""
		for w in words :
			result = result + Morse_encoder(w) + " "
		print(f"Morse Code is {result}")
		test_sent = input("Enter sentence (press Enter to process) >> ")
	print("Good bye ")
