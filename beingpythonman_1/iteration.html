Iteration

Motivation

Iteration หรือ การทำซ้ำ (repetition) เกิดขึ้นได้บ่อยในชีวิตจริง เช่น

  • การไปทำงานนอกบ้าน การรับประทานอาหาร
  • การผลิตสินค้าในโรงงาน
  • Apple ออกสินค้าหลายรุ่นใน model เดิม
  • การร้องเพลงเดิม ในหลายสถานที่
  • ฯลฯ
กิจกรรมเหล่านี้แม้เป็นการทำซ้ำ แต่หากมองในรายละเอียดแล้ว แต่ละรอบของกิจกรรมจะแตกต่างกันไม่มากก็น้อย

ในการสร้างโปรแกรมคอมพิวเตอร์ iteration ถือเป็นหัวใจหลักดวงหนึ่งของการสร้าง algorithms ถูกนิยามว่าเป็นกระบวนการทำงาน (process) ที่ต้องการให้ได้ผลลัพธ์ (outcome) แบบต่อเนื่องกัน (sequence of outcome) โดยผลลัพธ์ที่ได้จากแต่ละรอบการทำงานจะเป็นจุดเริมต้นสำหรับการทำงานรอบต่อไป [1]


การทำ iteration ในภาษา Python มี 2 แบบคือ

  1. For loops
  2. While loops

For Loops

ต้องอาศัย iterable object ได้แก่ List, String, Tuple,Set, Dictionary (ดูเพิ่มเติมเรื่อง Variable and Data Types) รูปแบบทั่วไปของชุดคำสั่งคือ 

for <looping-variable> in <iterable object> :
  <instructions block>
  ....

ในแต่ละรอบของ iteration สมาชิกจาก iterable object จะถูกเข้าถึงทีละตัว แล้วนำค่าไปกำหนดให้กับ looping variable จนครบ ดังนั้นจำนวนรอบของ for-loop มีจำนวนเท่ากับจำนวนสมาชิกของ interable object เช่น [2,9,0,"a"] มีสมาชิก 4 ตัว เมื่อนำมาทำ iteration ก็จะได้ 4 รอบการทำงาน

  • รอบ 1 : นำ 2 ออกไปไช้
  • รอบ 2 : นำ 9 ออกไปไช้
  • รอบ 3 : นำ 0 ออกไปไช้
  • รอบ 4 : นำ "a" ออกไปไช้

 Tip:

For-loop คือ difinite loop เนื่องจากสามารถบอกจำนวนรอบของการทำซ้ำได้ชัดเจน


ตัวอย่าง 1 : ใช้ String เป็น iteration object 

iter_obj = "Hello"            
for c in iter_obj :
	print(c)

 Output:

H
e
l
l
o

อธิบาย

  1. ได้กล่าวมาแล้วในเรื่อง Data type ว่า String มีเลข index ประจำในแต่ละตัวอักษรจัดเป็น iterable object
  2. ตัวแปร c เป็น looping variable ค่าของ c จะถูกกำหนดให้โดยอัตโนมัติด้วยตัวอักษรทีละตัวจาก iter_obj เรืิมจากซ้ายไปขวา
  3. เมื่อ c ได้รับค่าแล้ว ชุดคำสั่งที่ย่อหน้าถัดลงมาจะถูกเรียก ในตัวอย่างคือ print(c)
  4. หลังจากชุดคำสั่งทำงานครบแล้ว c ก็จะถูกกำหนดค่าด้วยตัวอักษรถัดไป
  5. โปรแกรมทำงานซ้ำในข้อ 3 - 4 ไปจนกว่าอักษรตัวสุดท้ายจะถูกใช้

  2. ตัวแปร c เป็น looping variable จะได้รับค่าเป็น 0,1,2,3 ตามลำดับ หมายถึงชุดคำสั่งภายใน for-loop จะถูกเรียกใช้ 4 รอบ
  3. ชุดคำสั่ง print(f"Hello {c}") คือ การสั่งให้พิมพ์ข้อความ "Hello " และค่าของ c ในรอบการทำงานปัจจุบัน

  2. ตัวแปร sum_age ใช้เก็บผลบวกของอายุ กำหนดค่าเริ่มต้นเป็น 0

  4. เมื่อรวมค่าอายุของสมาชิกครบแล้ว คำนวณหาค่าเฉลี่ยโดยนำค่าของ sum_age มาหารด้วยจำนวนข้อมูลในชุดข้อมูล

 Tip:

การใช้ range()

range() คือ built-in function ใช้สร้างลำดับของตัวเลขจำนวนเต็ม โดยสามารถกำหนดให้เป็นลำดับแบบเพิ่มค่าหรือลดค่าก็ได้ มีรูปแบบคือ 

range(start,stop,step)

การสร้างชุดของตัวเลข (r) จะใช้สูตร \(r[i] = start + step \times i \) โดย \(i \geq 0 \) และ \( r[i] < stop \) ในกรณีที่ step เป็นค่าบวก หรือ \( r[i] > stop \) ในกรณี step เป็นค่าลบ เช่น

range(1,5,2) หมายถึงการสร้างชุดของตัวเลข start = 1, stop < 5 และ step = 2 ลำดับที่ได้จะเป็นลำดับแบบเพิ่มค่า เพราะค่า step เป็นบวก
i\(r[i] = start + step \times i \)
0\(1 + 2 \times 0 = 1\)
1\(1 + 2 \times 1 = 3\)
2\(1 + 2 \times 2 = 5\)
ค่า \( r[2] = 5 \) จะไม่นำมาใช้เพราะมีค่าเท่ากับ stop ดังนั้นจะได้ชุดของตัวเลขเป็น [1,3]

range(10,0,-2) หมายถึงการสร้างชุดของตัวเลข start = 10, stop > 0 และ step = -2 ลำดับที่ได้จะเป็นลำดับแบบลดค่า เพราะค่า step เป็นลบ
i\(r[i] = start + step \times i \)
0\(10 + (-2 \times 0) = 10 \)
1\(10 + (-2 \times 1) = 8\)
2\(10 + (-2 \times 2) = 6\)
3\(10 + (-2 \times 3) = 4\)
4\(10 + (-2 \times 4) = 2\)
5\(10 + (-2 \times 5) = 0\)
ค่า \( r[5] = 0 \) จะไม่นำมาใช้เพราะมีค่าเท่ากับ stop ดังนั้นจะได้ชุดของตัวเลขเป็น [10,8,6,4,2]

ชุดของตัวเลขที่สร้างจาก range() จะไม่แสดงให้เห็น หากต้องการเห็นชุดของตัวเลขให้ใช้ list() เช่น 

r = range(1,10,2)
print(list(r))

 Output:

[1,3,5,7,9]

  Remember :

  1. ค่าที่ต้องระบุ (mandotory) ใน range() คือ stop
  2. หากไม่กำหนดค่า start จะใช้ค่าปริยาย (default) คือ 0
  3. หากไม่กำหนดค่า step จะใช้ค่าปริยาย (default) คือ 1
เช่น range(10) มีความหมายเดียวกับ range(0,10,1) (ดูเพิ่มเติม )

ตัวอย่าง 5 : การประมาณค่า \(\pi\)

การประมาณค่า \(\pi\) สามารถทำได้โดยการคำนวณจาก infinite series : \[\pi = 3 + \frac{4 (-1)^i}{(2+ 2i)(3+2i)(4+2i)}, 0 \leq i \leq \infty \] 

def create_element(i):
	dominator = (2+2*i)*(3+2*i)*(4+2*i)
	numerator = 4 * math.pow(-1,i)
	return numerator/dominator
	
def estimate_pi(l):
	s = 3
	for i in range(l):
		s  = s + create_element(i)
	return s

iterations = 20 # 20 rounds is enough for test 
est_pi = estimate_pi(iterations)
print(est_pi)

 Output:

3.1415

ตัวอย่าง 6 : Caesar Cipher

Caesar Cipher คือ algorithm เข้ารหัส (encryption) ที่คิดขึ้นมาโดย Julius Ceasar เพื่อใช้ปกปิดข้อความในคำสั่งไม่ให้คนอื่นอ่านเข้าใจ นอกจากคนที่รู้วิธีถอดรหัส (decryption) หลักการคือ ขยับตำแหน่งตัวษรไป 3 ตำแหน่ง เช่น ถ้าอักษรจริงคือ A อักษรที่จะถูกเขียนไปบนเอกสารจะ เป็นอักษร D อักษร N จะเป็น Q อักษร T จะเป็น W ดังนั้นคำ "ANT" จะถูกเขียนเป็น "DQW" เป็นต้น

ตัวอย่างในข้อนี่้จะแสดงการสร้าง look up table เพื่อโยงตัวอักษรต้นฉบับ A - Z กับอักษรที่จะใช้ในการเข้ารหัส 

def caesar_lookup_table():
	# ord("A") = 65, ord("Z") = 90
	lookup_tab = {}
	for i in range(65,91,1):
		org_chr = chr(i)
		enc_chr = chr(i+3)		
		lookup_tab[org_chr] = enc_chr
  lookup_tab[chr(32)] = chr(35) 
	return lookup_tab

print(caesar_lookup_table())

'I': 'L', 'J': 'M', 'K': 'N', 'L': 'O', 
'M': 'P', 'N': 'Q', 'O': 'R', 'P': 'S', 
'Q': 'T', 'R': 'U', 'S': 'V', 'T': 'W', 
'U': 'X', 'V': 'Y', 'W': 'Z', 'X': '[', 'Y': '\\',

 Tip:

  1. chr(i) เป็น Python built-in function ใช้เปลี่ยนจำนวนเต็ม i ไปเป็นอักษร Basic Latin
  2. รหัส Unicode ของอักษร A - Z คือ 65 - 90 , วรรค (space) คือ 32 และ "#" คือ 35

While Loops

While Loops ใช้ค่าความจริงของเงื่อนไข (condition) ที่กำหนดขึ้นแบบเดียวกับการใช้ if เป็นตัวชี้ว่าการทำซ้ำจะหยุดหรือดำเนินต่อ ทำให้ไม่สามารถกำหนดจำนวนรอบได้ชัดเจน จึงเรียกการทำซ้ำแบบนี้ว่า Indefinite loop รูปแบบทั่วไปคือ 

while <condition> :
  <instructions block>


ตัวอย่าง 1 : ประมาณค่ารากที่ 2 ด้วย Newton-Raphson method

Newton-Raphson method เป็น algorithm ใช้ประมาณค่ารากที่ 2 ของจำนวนจริงบวก a จากสูตร (สรุปจากเอกสารประกอบ) \[x_{n+1} = \frac{1}{2}(x_n + \frac{a}{x_n})\] เมื่อ \(x_{n+1} \) คือค่ารากที่ 2 ของ a โดยประมาณ

จะเห็นได้ว่า Algorithm นี้ต้องมีการคำนวณแบบ iteration จนกว่าจะได้ค่า \(x_{n+1} \) ที่เหมาะสม ซึ่งคำว่า "เหมาะสม" นั้นทำให้ ไม่สามารถระบุจำนวนรอบของการทำงานได้ชัดเจน ขึ้นกับเงื่อนไข (condition) ที่ตั้งขึ้น โดยในตัวอย่างจะให้ผู้ใช้กำหนดค่า threshold ขึ้นมาค่าหนึ่ง และความเหมาะสมจะเกิดขึ้นเมื่อ \(|x_{n+1} - a |< \text{threshold} \) 

def newton_method_sqrt(x,threshold=0.001):
  # initialize guess, any real number which greater than 0 is ok.
	guess = x/2   
  
	diff = x - guess*guess  
	while abs(diff) > threshold :
		guess = (x/guess + guess) * 0.5 # update guess
		diff = x - guess*guess
		
	return guess	 
  
# test 1
threshold = 0.00001
a = 9
sqrt = newton_method_sqrt(a,threshold)
print(f"Square root of {a} is {sqrt}")
  
# test 2
threshold = 0.00001
a = 50
sqrt = newton_method_sqrt(a,threshold)
print(f"Square root of {a} is {sqrt}")
  
# test 3
threshold = 0.00001
a = 12.56
sqrt = newton_method_sqrt(a,threshold)
print(f"Square root of {a} is {sqrt}")

 Output:

Square root of 9 is 3.0000000000393214
Square root of 50 is 7.071067928984419
Square root of 12.56 is 3.5440090386179595

 Tip:

abs(x) เป็น Python built-in function ทางคณิตศาสตร์ จะส่งค่า absolute value ของจำนวนจริง x ออกมา เช่น abs(-5) = 5

ตัวอย่าง 2 หา Greatest commond divisor ด้วยวิธีของ Euclidean

Greatest commond divisor (GCD) ของสองจำนวน M,N คือจำนวนที่มีค่ามากที่สุดที่หาร M,N ลงตัว เช่น 12,6 จะมี GCD คือ 6 ตัวอย่างนี้แสดงให้เห็นนำเอา Euclidean algorithm มาใช้ 

def gcd(m,n):	
	while not(m==0 or n ==0):		
		mi = min(m,n)
		mx = max(m,n)
		r = mx % mi
		m = mi
		n = r

	if m == 0 :
		return n
	if n == 0 :
		return m
		        
print(gcd(10,12))
print(gcd(192,270))

 Output:

2
6

 Tip:

  1. min(x,y) เป็น Python built-in function ทางคณิตศาสตร์ จะส่งค่าที่น้อยที่สุดระหว่าง x,y
  2. max(x,y) เป็น Python built-in function ทางคณิตศาสตร์ จะส่งค่าที่มากทีสุดระหว่าง x,y

Break และ Continue

เหตุการณ์ในชีวิตจริง เช่น การประชาสัมพันธ์ส่งเสริมการขายสินค้า A โดยจะทำการลดราคา 50 % จากราคาขายตลอดเดือนตุลาคมหรือจนกว่า สินค้าจะหมด ความคาดหวังคือจะ ทำการลดราคาตลอด 31 วัน แต่หากสินค้าหมดก่อนก็ต้องหยุด (break) หรือหากสามารถหาสินค้ามาเติมได้ก็จะดำเนินการต่อจนครบ (continue) การที่สินค้าหมดลงหรือ การมีสินค้ามาเพิ่มเป็นเงื่อนไขที่อาจเกิดขึ้นแล้วทำให้การดำเนินไปของกิจกรรมเปลี่ยนไป เหตุการณ์ทำนองนี้เกิดขึ้นได้ในการเขียนโปรแกรมคอมพิวเตอร์ด้วยเช่นกัน ในบางกรณี การตรวจสอบเงื่อนไขภายใน iteration body เองอาจจะส่งผลให้การดำเนินไปของ loop ไม่เป็นไปตามปรกติ ในภาษา Python มีคำสั่งที่ใช้คือ break และ continue

การทำงานของ break statement อธิบายตามที่แสดงในรูปข้างล่างนี้


continue statement

continue เป็นคำสั่งที่ให้ข้ามชุดคำสั่งที่ตามหลังทั้งหมดใน iteration body แล้วเดินหน้าไปสู่การทำงานรอบถัดไป 

for x in range(0,5,1) :
	print("before continue")
	if x == 3 :
	    # skip to next iteration
		continue
	print(x)		
	print("----")
# outside the loop
print("Outside loop body")

 Output:

before continue
0
----
before continue
1
----
before continue
2
----
before continue
before continue
4
----
Outside loop body

ตัวอย่างนี้แสดงให้เห็นว่ามีการข้ามไม่ได้ทำชุดคำสั่งที่อยู่ตามหลัง continue เมื่อ x มีค่าเป็น 3 ตามเงื่อนไข การทำงานของ continue statement อธิบายตามที่แสดงในรูปข้างล่างนี้


S = "1234*56789"
i  = 0 
while i < len(S) :	
	if S[i] == "*" :
		continue
	print(S[i])
	i = i+1

 Output:

1
2
3
4

ตัวอย่างนี้แสดงให้เห็นจุดที่ต้องระวังในการใช้งาน continue หลังจากที่การตรวจสอบเงื่อนไข S[i] == "*" เป็นจริง คำสั่งที่ตามหลังจะถูกข้ามทั้งหมด รวมทั้งคำสั่งที่เปลี่ยนค่าของ i ด้วย เมื่อค่า i ไม่เปลี่ยน ค่าของ S[i] ก็ไม่เปลี่ยนด้วย ผลก็คือชุดคำสั่งนี้จะทำงาน ไม่รู้จบ (endless loop หรือ infinite loop)

List comprehension

ยกตัวอย่าง ถ้ามี X = [1,2,3,4,5] และต้องการสร้าง List ที่เป็นกำลังสองของสมาชิกใน X โดยทั่วไปอาจเขียนแบบนี้ 

X = [1,2,3,4,5]
Y = []
for i in X:
  Y.append(i**2)
print(Y)

 Output:

2

หากเขียนด้วย comprehension จะได้เป็น 
X = [1,2,3,4,5]
Y = [i**2 for i in X]
print(Y)

 Output:

 [1,4,9,16,25]

ให้สังเกตุชุดคำสั่ง Y = [i**2 for i in X]
  • i**2 คือ output
  • และ for i in X คือ input
  • ไม่มีการกำหนด condition

ทดลองเพิ่มเงื่อนไข เช่น ต้องการค่ากำลังสองเฉพาะที่เป็นเลขคี่ 
X = [1,2,3,4,5]
Y = [i**2 for i in X if i % 2 != 0]
print(Y)

 Output:

 [1,9,25]

คราวนี้ชุดคำสังได้มีการเพิ่ม if i % 2 != 0 เข้าไปทำให้เลือกเอาแต่เลขคี่ไปทำงานต่อ


Dictionary comprehension

ใช้หลักการเดียวกันกับ List comprehension เช่น {"John":60,"Mark":40} คือ dictionary มี key เป็นชื่อคนและ value คืออายุมีหน่วยปี
ถ้าต้องการสร้าง dictionary ที่เก็บ ชื่อและอายุในหน่วยของเดือน ก็อาจเขียนได้เป็น 

DY = {"John":60,"Mark":40}
DM = {key:val * 12 for key,val in DY.items()  }
print(DM)

 Output:

{'John': 720, 'Mark': 480}

10. การคำนวณหาเกรดเฉลี่ย (Grade Point Average, GPA) ได้จากสูตร \[ GPA = \frac{\sum_{i=1}^{n}G_iC_i}{\sum_{i=1}^{n}C_i} \] เมื่อ

  • \(G_i\) คือ เกรดของวิชาที่ i
  • \(C_i\) คือ จำนวนหน่วยกิตของวิชาที่ i
