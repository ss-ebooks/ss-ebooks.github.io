รู้จักกับ C programming language

คุณลักษณะทั่วไป (Features)

C เป็น programming language.พัฒนาขึ้นโดย Dennis Ritchie ในปี ค.ศ. 1972มีเป้าหมายในตอนแรกเพื่อใช้เป็น system programming language เพื่อใช้สร้างซอฟต์แวร์ระบบปฏิบัติการ (operating system) มีคุณสมบัติที่สำคัญดังนี้

Procedural language
ในโปรแกรม 1 โปรแกรมอาจถูกแบ่งออกเป็นการทำงานย่อยได้มากกว่า 1 งาน แต่ละงานมีเป้าหมายเดียว เรียกว่าแต่ละงานว่า procedure หรือ function
ความรวดเร้วในการทำงาน
C programming language จัดเป็น middle-level language ช่วยให้ programmers เข้าถึงการทำงานของ computer hardware โดยตรง ในขณะที่ higher-level languages เช่น Python ไม่สามารถทำได้.
