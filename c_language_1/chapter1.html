รู้จักกับ C programming language

บทนำ (Introduction)

C เป็น programming language.พัฒนาขึ้นโดย Dennis Ritchie ในปี ค.ศ. 1972มีเป้าหมายในตอนแรกเพื่อใช้เป็น system programming language เพื่อใช้สร้างซอฟต์แวร์ระบบปฏิบัติการ (operating system) ปัจจุบัน International Standardization Organization (ISO) เป็นผู้กำหนดมาตรฐานสำหรับภาษา C เรียกว่า C Standard ล่าสุดคือ C17 (บางครั้งอาจเรียกว่า C18)C Standard รุ่นก่อนหน้าคือ C11

ข้อกำหนดที่สร้างขึ้นจะเป็นตัวกำหนดรูปแบบการทำงานของโปรแกรมและวิธีการเขียนชุดคำสั่ง ยกตัวอย่างเช่น 

The function called at program startup is named main. The implementation
declares no prototype for this function. It shall be defined with a return
type of int and with no parameters:
int main(void) { /* ... */ }
or with two parameters (referred to here as argc and argv, though any names
may be used, as they are local to the function in which they are declared):
int main(int argc, char *argv[]) { /* ... */ }
or equivalent;10) or in some other implementation-defined manner.

ข้อความนี้กำหนดว่า โปรแกรมที่เขียนด้วย C language จะตัองมี function ชื่อ main() เสมอ เพราะพฤติกรรมของโปรแกรมจะเริ่มต้นการทำงานจาก function นี้ซึ่ง main สามารถเขียนได้ใน 2 รูปแบบคือ แบบมี 2 parameters และไม่มี parameters เป็นต้น

C language อาจถูกมองว่าเป็นภาษาที่ค่อนข้างยากสำหรับผู้เริ่มต้น แต่หากมองอีกมุมคือ C language ถูกใช้งานมาระยะยาวนานแล้ว มีภาษารุ่นใหม่หลายภาษาได้ยืม Syntax ของ C language ไปใช้ เช่น Java, JavaScript, PHP ฯล และมีหลายภาษาที่ใช้ C ในการพัฒนา library เช่น Python นอกจากนี้ C++ language ซ่งเป็น superset ของ C ก็มีการใช้งานแพร่หลายมากในปัจจุบัน Machine Learning Framework อย่าง TensorFlow ก็ใช้ C++ language ในการพัฒนา ดังนั้นการเริ่มต้นด้วย C language น่าจะเป็นผลดีในระยะยาว ช่วยให้การศึกษาภาษาอื่นง่ายขึ้น

โครงสร้างของ C program

โครงสร้างหลักของ C program มี 3 องค์ประกอบหลักคือ

  • Headers
  • Functions
  • main function

Headers
เป็นพืนที่ของสิ่งที่เรียก่ว่า pre-processor directivesชุดคำสั่งของส่วนนี้จะถูกจัดเก็บแยกออกจากโปรแกรมที่กำลังใช้งาน และจะถูกนำมาประมวลผลก่อนส่วนอื่น แต่ละบรรทัดจะมีเครื่องหมาย # เป็นอักษรตัวแรกและไม่มี ; ปิดท้ายบรรทัด ที่มักพบบ่อยได้แก่

  • #include
  • #define
  • #ifdef
  • #endif
  • #if
  • #else

Pre-processor แบ่งได้ 3 ประเภทได้แก่

File inclusion
รูปแบบการเขียน 
#include “filename” หรือ #include <filename>

เช่น 
#include “stdio.h”
#include <opencv4/higui.h>

Macro
รูปแบบการเขียน 
#define identifier string

เช่น 
#define  MAX_NUM 2000
#define sqrt(x) x*x

Compiler Control
ใช้กำหนดเงื่อนไขในการ compile โปรแกรม จะขอข้ามเรื่องนี้ไปก่อน ในระดับของการเริ่มต้นไม่จำเป็นต้องทราบในรายละเอียดเรื่องนี้

Functions
เป็นพืนที่ของการกำหนดหน้าที่ต่างๆ ในโปรแกรม ซึ่งจะกล่าวถึงต่อไป

Main Function
เป็นจุดเริ่มต้นการทำงานของโปรแกรมใน C language มีจุดสำคัญคือ

  • ชื่อของ function ต้องใช้ main()
  • ต้อง return ค่าออกมาเป็น integer
  • หากรับ parameters จะใช้ int argc และ char *argv[] (จะอธิบายความหมายภายหลัง) หากไม่มีการรับ parameter ให้ใช้ void
ตัวอย่างเช่น 
int main(void){
printf("Hello World");
return 0;
}


int main(int argc, char *argv[]){
printf("Hello World");
return 0;
}

C Tokens

Token หมายถึงองค์ประกอบที่เล็กที่สุดแต่ยังคงมีความหมายสำหรับ compiler สำหรับ C language แบ่ง tokes ออกได้ 6 ประเภท ได้แก่

  1. Keywords (Reserved words)
  2. Identifiers
  3. Constants
  4. Strings
  5. Special symbols
  6. Operators

Keywords
หมายถึงคำที่ถูกระบุความหมายหรือหน้าที่ไว้แล้ว ไม่สามารถนำไปใช้เป็นชื่อสำหรับตัวแปรได้ ไม่สามารถกำหนดความหมายหรือหน้าที่ได้แต่สามารถที่จะหาคำมาทดแทนได้ โดยการใช้ประโยชน์จาก compile control directive ที่กล่าวไว้ในตอนก่อนหน้า keywords ใน C language มี 32 keywords คือ

autodoubleintstruct
breakelselongswitch
caseenumregistertypedef
charexternreturnunion
constfloat shortunsigned
continueforsignedvoid
defaultgotosizeofvolatile
doifstaticwhile

Identifiers
หมายถึงคำที่นำมาใช้เป็นชื่อของตัวแปร หรือ function กฏในการใช้ identifier มีดังนี้

  • อักษรตัวแรกต้องเป็นอักษรในภาษาอังกฤษ หรือ underscore(_)
  • สามารถใช้ตัวอักษรภาษาอังกฤษ ตัวเลข (0-9) และ underscore(_) ร่วมกันได้ ห้ามใช้ตัวอักษรพิเศษ
  • ต้องไม่ใช่่ keywords
  • ไม่ควรมีความยาวมากกว่า 31 ตัวอักษร

Constants
บางครั้งถูกเรียก่ว่า literal คือตัวแปรที่ไม่สามารถเปลี่ยนแปลงค่าที่กำหนดให้ตั้งแต่แรกได้ การประกาศจะใช้ const นำหน้าชื่อ

String
คือชุดของตัวอักษรที่มีอักขระพิเศษปิดท้าย และต้องถูกล้อมด้วย "" เช่น "Hello World"

Special symbols
คืออักขระพิเศษที่มีความหมายในตัว ได้แก่

สัญญลักษณ์ความหมาย
Brackets [ ] ใช้ระบุค่าของตัวแปร array
Parentheses ()function call
Braces {}code block
Comma (, )แยกชุดคำสั่ง
Semicolor ; จบบรรทัดของชุดคำสั่่ง
Asterisk * สร้างตัวแปรประเภท poiter (กล่าวถึงภายหลัง)
Pre-processor # กล่าวถึงในตอนก่อนหน้า

Operators
แบ่งได้ 3 ประเภท

Unary operator
Operator ที่ต้องการเพียง 1 operand เช่น a++ , ++b เป็นการสั่งให้เพิ่มค่าของ a และ b ขึ้น 1 โดย a และ b คือ operand และ ++ คือ operator
Binary operator
Operator ที่ต้องการ 2 operands เช่น a = 5, b = 6 เป็นการกำหนดค่าให้ a,b โดย a,b,5,6 คือ operands และ = คือ operator
Ternary operator
มักรู้จักในชื่อ condition operator สัญญลักษณ์คือ ? รูปแบบการใช้งาน 
expression 1 ? expression 2 : expression 3
