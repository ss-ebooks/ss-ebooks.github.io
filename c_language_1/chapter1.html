รู้จักกับ C programming language

บทนำ (Introduction)

C เป็น programming language.พัฒนาขึ้นโดย Dennis Ritchie ในปี ค.ศ. 1972มีเป้าหมายในตอนแรกเพื่อใช้เป็น system programming language เพื่อใช้สร้างซอฟต์แวร์ระบบปฏิบัติการ (operating system) ปัจจุบัน International Standardization Organization (ISO) เป็นผู้กำหนดมาตรฐานสำหรับภาษา C เรียกว่า C Standard ล่าสุดคือ C17 (บางครั้งอาจเรียกว่า C18)C Standard รุ่นก่อนหน้าคือ C11

ข้อกำหนดที่สร้างขึ้นจะเป็นตัวกำหนดรูปแบบการทำงานของโปรแกรมและวิธีการเขียนชุดคำสั่ง ยกตัวอย่างเช่น 

The function called at program startup is named main. The implementation
declares no prototype for this function. It shall be defined with a return
type of int and with no parameters:
int main(void) { /* ... */ }
or with two parameters (referred to here as argc and argv, though any names
may be used, as they are local to the function in which they are declared):
int main(int argc, char *argv[]) { /* ... */ }
or equivalent;10) or in some other implementation-defined manner.

ข้อความนี้กำหนดว่า โปรแกรมที่เขียนด้วย C language จะตัองมี function ชื่อ main() เสมอ เพราะพฤติกรรมของโปรแกรมจะเริ่มต้นการทำงานจาก function นี้ซึ่ง main สามารถเขียนได้ใน 2 รูปแบบคือ แบบมี 2 parameters และไม่มี parameters เป็นต้น

C language อาจถูกมองว่าเป็นภาษาที่ค่อนข้างยากสำหรับผู้เริ่มต้น แต่หากมองอีกมุมคือ C language ถูกใช้งานมาระยะยาวนานแล้ว มีภาษารุ่นใหม่หลายภาษาได้ยืม Syntax ของ C language ไปใช้ เช่น Java, JavaScript, PHP ฯล และมีหลายภาษาที่ใช้ C ในการพัฒนา library เช่น Python นอกจากนี้ C++ language ซ่งเป็น superset ของ C ก็มีการใช้งานแพร่หลายมากในปัจจุบัน Machine Learning Framework อย่าง TensorFlow ก็ใช้ C++ language ในการพัฒนา ดังนั้นการเริ่มต้นด้วย C language น่าจะเป็นผลดีในระยะยาว ช่วยให้การศึกษาภาษาอื่นง่ายขึ้น

โครงสร้างของ C program


Statements
ในมุมมองหนึ่ง โปรแกรมคอมพิวเตอร์คือกลุ่มของ statments ที่ถูกจัดลำดับเพื่อกำหนดให้คอมพิวเตอร์ทำงานเพื่อบรรลุเป้าหมายอย่างใดอย่างหนึ่ง ดังนั้น statements จึงถูกจัดว่าเป็นชุดคำสั่งแบบหนึ่ง statements ในคอมพิวเตอร์โปรแกรมทำหน้าที่เหมือนกับ statments ในภาษาธรรมชาติ (natural language) C language กำหนดให้ใช้เครื่องหมาย " ;" เพื่อแสดงการจบ statment เช่นกับ "." ใช้จบ statement ในภาษาอังกฤษ

Statement ใน C language แบ่งออกได้หลายชนิด (รายละเอียดจะกล่าวถึงตอนหลัง) ได้แก่

  • Declaration statements
  • Jump statements
  • Expression statements
  • Compound statements
  • Selection statements (conditionals)
  • Iteration statements (loops)
  • Try blocks

เช่นเดียวกับการเขียนเรียงความ โครงสร้างหลักของ C program มี 3 องค์ประกอบหลักคือ

  • Headers
  • Functions
  • main function

Headers
เป็นพืนที่ของสิ่งที่เรียก่ว่า pre-processor directivesชุดคำสั่งของส่วนนี้จะถูกจัดเก็บแยกออกจากโปรแกรมที่กำลังใช้งาน และจะถูกนำมาประมวลผลก่อนส่วนอื่น แต่ละบรรทัดจะมีเครื่องหมาย # เป็นอักษรตัวแรกและไม่มี ; ปิดท้ายบรรทัด ที่มักพบบ่อยได้แก่

  • #include
  • #define
  • #ifdef
  • #endif
  • #if
  • #else

Pre-processor แบ่งได้ 3 ประเภทได้แก่

File inclusion
รูปแบบการเขียน 
#include “filename” หรือ #include <filename>

เช่น 
#include “stdio.h”
#include <opencv4/higui.h>

Macro
รูปแบบการเขียน 
#define identifier string

เช่น 
#define  MAX_NUM 2000
#define sqrt(x) x*x

Compiler Control
ใช้กำหนดเงื่อนไขในการ compile โปรแกรม จะขอข้ามเรื่องนี้ไปก่อน ในระดับของการเริ่มต้นไม่จำเป็นต้องทราบในรายละเอียดเรื่องนี้


Functions
เป็นพืนที่ของการกำหนดหน้าที่ต่างๆ ในโปรแกรม ซึ่งจะกล่าวถึงต่อไป

Main Function
เป็นจุดเริ่มต้นการทำงานของโปรแกรมใน C language มีจุดสำคัญคือ

  • ชื่อของ function ต้องใช้ main()
  • ต้อง return ค่าออกมาเป็น integer
  • หากรับ parameters จะใช้ int argc และ char *argv[] (จะอธิบายความหมายภายหลัง) หากไม่มีการรับ parameter ให้ใช้ void
ตัวอย่างเช่น 
int main(void){
    printf("Hello World");
    return 0;
}


int main(int argc, char *argv[]){
    printf("Hello World");
    return 0;
}

 Tip:

โปรแกรมทุกโปรแกรมใน C langauge จะต้องมี main() เสมอ โปรแกรมจะเริ่มทำงานใน statement แรกและจบการทำงานหลังจากทำงาน statement สุดท้ายใน main() เสร็จ

C Tokens

Token หมายถึงองค์ประกอบที่เล็กที่สุดแต่ยังคงมีความหมายสำหรับ compiler สำหรับ C language แบ่ง tokes ออกได้ 6 ประเภท ได้แก่

  1. Keywords (Reserved words)
  2. Identifiers
  3. Constants
  4. Strings
  5. Special symbols
  6. Operators

Identifiers
หมายถึงคำที่นำมาใช้เป็นชื่อของตัวแปร หรือ function กฏในการใช้ identifier มีดังนี้

  • อักษรตัวแรกต้องเป็นอักษรในภาษาอังกฤษ หรือ underscore(_)
  • สามารถใช้ตัวอักษรภาษาอังกฤษ ตัวเลข (0-9) และ underscore(_) ร่วมกันได้ ห้ามใช้ตัวอักษรพิเศษ
  • ต้องไม่ใช่่ keywords

Constants
บางครั้งถูกเรียก่ว่า literal คือตัวแปรที่ไม่สามารถเปลี่ยนแปลงค่าที่กำหนดให้ตั้งแต่แรกได้ การประกาศจะใช้ const นำหน้าชื่อ

Operators
แบ่งได้ 3 ประเภท

Unary operator
Operator ที่ต้องการเพียง 1 operand เช่น a++ , ++b เป็นการสั่งให้เพิ่มค่าของ a และ b ขึ้น 1 โดย a และ b คือ operand และ ++ คือ operator
Binary operator
Operator ที่ต้องการ 2 operands เช่น a = 5, b = 6 เป็นการกำหนดค่าให้ a,b โดย a,b,5,6 คือ operands และ = คือ operator
Ternary operator
มักรู้จักในชื่อ condition operator สัญญลักษณ์คือ ? รูปแบบการใช้งาน 
expression 1 ? expression 2 : expression 3

การประกาศตัวแปร (Varaiable Declaration)

ตัวแปร (Variable) เป็นส่วนสำคํญที่ต้องถูกนำมาใช้ตลอดเวลาในการเขียนคอมพิวเตอร์ในทุกภาษา ยกตัวอย่างเช่น การหาพื้นที่วงกลมจากสูตร \( area = \pi r^2 \) ต้องใช้ variable 2 ตัวเพื่อใช้เก็บค่าของข้อมูลที่เป็นพื้นที่ (area) และรัศมีของวงกลม (r) ส่วนค่า \(\pi\) ก็จัดเป็น variable แบบหนึ่งแต่เป็น variable ที่ไม่เปลี่ยนแปลงค่า (constant)

ตัวแปรในโปรแกรมคือสิ่งที่ใช้ตำแหน่งในหน่วยความจำหลักของคอมพิวเตอร์ที่ใช้จัดเก็บค่าของตัวแปรที่กำลังอ้างถึง จินตนาการถึงหนังสือ แต่ละหน้าของหนังสือเทียบได้กับตัวแปร แต่ละหน้ามีข้อความเทียบได้กับค่าของข้อมูลและเลขหน้าระบุตำแหน่งของหน้าในหนังสือซึ่งเทียบได้กับหน่วยความจำหลักทั้งหมด

การสร้างตัวแปรจะอาศัย statement แบบพิเศษเรียกว่า definitionรูปแบบทั่วไปของ statement สำหรับการสร้างตัวแปรคือ

<datatype> <variable name> = <initializing value>

การประกาศตัวแปรทำได้ใน 2 รูปแบบคือ

  1. การประกาศแบบไม่มีค่าเริ่มต้น (without iniitializing value)
  2. การประกาศแบบมีค่าเริ่มต้น (with iniitializing value)

เช่น

int x = 20;

เป็นการประกาศว่า ตัวแปร x มี datatype เป็น integer ใช้ขนาดพื้นที่ 2 bytes ค่าที่เป็นไปได้อยุ่ระหว่าง -32,768 ถึง 32,767 ค่าปัจจุบันคือ 20ซึ่งเป็นตัวอย่างของการประกาศตัวแปรโดยมีค่าเริ่มต้น เปรียบได้กับหนังสือที่มีเลขหน้าและข้อความบนหน้ากระดาษแล้ว

อีกรูปแบบหนึ่ง เช่น

int y ;

การประกาศแบบนี้ ตัวแปร y ยังไม่มีค่าในตัวเทียบได้กับกระดาษเปล่า ยังไม่มีเลขหน้า ก่อนนำไปใช้งานต้องทำการให้ค่าเสียก่อน เมื่อตัวแปรได้รับค่าแล้วก็จะเหมือนกับกระดาษที่มีข้อความและมีเลขหน้า

 Tip:

C อนุญาตให้สามารถประกาศตัวแปรหลายตัวที่มี datatype เดียวกันในบรรทัดเดียวกันได้ เช่น
int a=2,b=3,c=5;

  Remember :

กฏการประกาศตัวแปร
  1. ตัวแปรจะต้องไม่มีชื่อซ้ำกันในโปรแกรมเดียวกัน
  2. ตัวแปรจะเป็นได้เพียง 1 datatype เท่านั้น
  3. ตัวแปรจะไม่สามารถเปลี่ยนชนิดของ datatype ได้ ตลอดการทำงานของโปรแกรม

Compiler, Linker, Libraries

พิจารณาขั้นตอนการพัฒนาโปรแกรมด้วย C language

ในขั้นตอน compile programm คือการแปลงชุดคำสั่ง C language ที่เขียนขึ้นไปเป็นภาษาเครื่อง "machain language "สิ่งที่ต้องใช้คือ " ตัวแปรภาษา" (compiler) โดย compiler จะทำการตรวจสอบ statement ทีละ statement ซึ่งประกอบด้วยกิจกรรมหลัก 2 อย่างคือ

  1. ตรวจสอบความถูกต้องของการเขียน statement ว่าตรงตามกฏของภาษา (syntax) หรือไม่ หากพบความผิดพลาดจะบันทึกไว้ แล้วทำการรายงานออกมาหลังจากการตรวจสอบ statement สุดท้ายเสร็จ ในขั้นตอนนี้ยังไม่มีการแปลงภาษา
  2. หากไม่พบข้อผิดพลาดใด จะทำการแปลง statements ให้เป็น machine language รูปแบบที่เรียกว่า object file โดยทั่วไป object file คือ file ที่มีชื่อเดียวกับไฟล์ชุดคำสั่ง แต่มีนามสกุล ".o" หรือ ".obj " เช่น abc.c หลังจาก compile แล้ว compiler จะสร้าง object file ชื่อ abc.o ออกมา

    3. ขั้นตอนต่อไปคือการทำงานของ software ที่เรียกว่า linker งานที่ linker ทำมี 3 ขั้นตอนคือ

    1. ทำการรวบ object file(s) ที่ถูกสร้างขึ้นรวมเป็น 1 ไฟล์ เรียกว่า executable file
    2. ทำการเชื่อมโยง external library files หากมีการร้องขอในโปรแกรมชุดคำสั่ง
    3. แก้ไชปัญหาการเกิดความขัดแย้งของการใช้ libraries (จะกล่าวถึงภายหลัง)

First program : Area of a circle

เปิดหน้า Code editor and compiler แล้วพิมพ์ชุดคำสั่งตามตัวอย่าง

#include "stdio.h"

float process(int r){
    const float pi = 3.1416;
    float area = pi * r * r;
    return area;
}

int main() {
    int r  = 10;
    float area = process(r);
    printf("area of circle = %7.2f",area);
    return 0;
}

กดปุ่ม Run แล้วสังเกตุผลที่ได้

จากเรื่องโครงสร้างของโปรแกรม จะเห็นว่าโปรแกรมนี้ทั้ง 3 ส่วนคือ headers, functions และ main function ตามภาพข้างล่าง

ส่วน header มี pre-processor directive แบบ include file สั่งให้นำเนื้อหาหรือชุดคำสั่งใน stdio.h เข้ามา stdio มาจากคำว่า " standard input output" ภายในบรรจุคำสั่งที่ใช้สำหรับการแสดงผลและรับค่าจากผู้ใช้ ในตัวอย่างได้นำ prinf() มาใช้แสดงผล

ถัดลงมาเป็นส่วนของ functions ในตัวอย่างมี 1 function ชื่อ process() รับ 1 parameter เป็น data type แบบ integer ให้ชื่อตัวแปรเป็น r ภายใน process เป็นชุดคำสั่งสำหรับคำนวณหาพื้นที่วงกลม โดยนำค่า r ที่ได้มาคำนวณ แล้วส่งค่าออกไปจาก function ด้วยคำสั่ง return ค่าที่ส่งออกไปเป็นdata type แบบ float เพราะเมื่อนำค่า pi มาคูณกับค่า r แล้วผลลัพธ์จะต้องเป็นตัวเลขที่มีจุดทศนิยม จึ่งต้องกำหนด datat type ให้เป็น float หรือ double

ส่วนสุดท้ายคือ main function ซึ่งบรรจุลำดับการทำงานของโปรแกรมทั้งหมดไว้ เริ่มจากกำหนดค่า r ให้เป็น 10 แล้วนำไปคำนวณหาพื้นที่ด้วยการส่ง r ไปให้ process() จากนั้นนำค่าที่ได้มาแสดงผลด้วยคำสั่ง printf()
