วาด B-Spline curve ด้วย raylib

โปรแกรมนี้จะใช้สมการ 1.6 จากเรื่อง B-spline curve เป็น algorithm ในการวาด Cubic B-spline Curve โดยกำหนดให้ control points คือ (200,100),(300,300),(400,100),(500,400) ขั้นตอนทั้งหมดจะเหมือนกับการสร้าง Bezier curve ต่างกันตรงที่การสร้าง coefficient matrix

การสร้างชุดคำสั่ง

  1. กำหนด header files โดยมีการเพิ่ม header file ของ armadillo เข้ามา
#include "raylib.h"
#include "armadillo"
#include "vector"
  1. ประกาศตัวแปร drawPoints เป็นประเภทโครงสร้าง vector ของ Vector2 ไว้เก็บค่า coordinates ที่คำนวณได้
std::vector<Vector2> drawPoints;
  1. การทำ Interpolation อยู่ใน setup() 
void setup(){
	/* TODO : initialize all variables */
	
	/* create coefficient matrix */
	arma::mat M  = arma::mat("-1 3 -3 1;3 -6 0 4 ;-3 3 3 1;1 0 0 0");
	M = M/6;
	arma::mat P  = arma::mat("200 300 400 500;100 300 100 400");
	
	for(double t=0;t<=1;t+=0.01){
		arma::mat T(4,1);
		arma::mat St(2,1);
		T(3,0) = 1;
		T(2,0) = t;
		T(1,0) = t*t;
		T(0,0) = t*t*t; 
		St = P*M*T;
		drawPoints.push_back((Vector2){(float)St(0,0),(float)St(1,0)});
	}
}
  1. วาดเส้นโค้ง และ control points
