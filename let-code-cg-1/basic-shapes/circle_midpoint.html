Midpoint circle algorithm

การตรวจสอบว่าจุดอยู่ในหรือนอกวงกลม

ถ้ามีวงลมรัศมี r แล้วการจะบอกว่าจุด (x,y) ใดๆอยู่ในหรือนอกวงกลม ทำได้โดยการคำนวณหาค่าระยะทางของจุดๆนั้นกับจุดศูนย์กลาง

  • ถ้าระยะทางมากกว่ารัศมี แสดงว่าจุดนั้นอยู่นอกวงกลม
  • ถ้าระยะทางน้อยกว่ารัศมี แสดงว่าจุดนั้นอยู่ในวงกลม

การคำนวณ :

  • r1​ มีค่าน้อยกว่า r นั่นคือ r12​x12​+y12​∴r2−x12​−y12​​<r2<r2>0​(1.0)​

  • r2​ มีค่ามากกว่า r นั่นคือ r22​x22​+y22​∴r2−x22​−y22​​>r2>r2<0​(1.1)​

จาก (1.0),(1.1) ทำให้สรุปตัวแปรที่ใช้ตัดสินว่าจุด P(x,y) อยู่ในหรือนอกวงกลม Pk​=r2−x2−y2(1.2)

Pk​=⎩⎨⎧​>0<00​,(x,y) is in circle,(x,y) is outside circle,(x,y) is on circumference​

การทำงานของ Midpoint circle algorithm

  1. ณ จุด(x,y) ใดๆ ที่ถูกวาดไปแล้ว จุดต่อไปที่ทำการวาดจะเป็น (x,y+1) หรือ (x-1,y+1)

  2. หาจุดกึ่งกลางระหว่าง (x,y+1) และ (x-1,y+1) (2x+(x−1)​,2(y+1)+(y+1)​)=(x−0.5,y+1)=Pk​

  3. นำค่า Pk​ คำนวณค่าตามสมการที่ (1.2)

    • ถ้า Pk​>0 เลือก plot ที่จุด (x,y+1)
    • ถ้า Pk​≤0 เลือก plot ที่จุด (x-1,y+1)

  4. แบ่งพื้นที่เส้นรอบวงออกเป็น 8 ส่วน (octants) ตามรูปที่ 2 คำนวณหาตำแหน่ง pixel ทีจะถูก plot เพียงแต่ใน octant 1 ที่เหลือกใช้ความสัมพันธ์ที่แสดงตามรูปที่ 2

flowchart TD;
start-->id1(รับค่า cx,cy,r,color);
id1-->id2(x = r\ny = 0\nd = 1-r);
id2-->id3(plot at x,y);
id3-->id4(y = y + 1);
id4-->id5{ d > 0 ?};
id5--Yes -->id6(x = x - 1);
id6-->id7(d = d + 2y - 2x + 1);
id7-->id8(y = y + 1);
id5--No -->id9(d = d + 2y + 1);
id9-->id8;
id8-->id10{x >= y ?};
id10--Yes -->id3;
id10--No -->stop;

  4. ใส่ชุดคำสั่งให้กับ function ในข้อ 3 โดยอิงข้อมูลในรูปที่ 2 และ flowchart

    void plot(int cx,int cy,int x,int y,Color clr){
	DrawPixel(cx+x,cy+y,clr);// octant 1
	DrawPixel(cx+y,cy+x,clr);// octant 2
	DrawPixel(cx-y,cy+x,clr);// octant 3
	DrawPixel(cx-x,cy+y,clr);// octant 4
	DrawPixel(cx-x,cy-y,clr);// octant 5
	DrawPixel(cx-y,cy-x,clr);// octant 6
	DrawPixel(cx+y,cy-x,clr);// octant 7
	DrawPixel(cx+x,cy-y,clr);// octant 8
}

void draw_circle(int cx,int cy,int radius ,Color clr){
	int x = radius;
	int y = 0;
	int d = radius - 1;
	while(x>=y){
		plot(cx,cy,x,y,clr);
		y++;
		if(d > 0){
			x = x - 1;
			d = d + 2*y - 2*x + 1;
		}else{
			d = d + 2*y + 1;
		}
	}
}

  5. เพิ่มชุดคำสั่งในฟังก์ชั่น draw()

    void draw(){
	int radius = 200;
	draw_circle(cx,cy,radius,BLUE);
}
